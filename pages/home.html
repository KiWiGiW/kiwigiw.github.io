Projects

3Park

3Park platform represents a spatial data infrastructure designed to consolidate the geographical information of clients, including cells and parks, into a single comprehensive database. This solution facilitates the efficient visualization and management of containers within the park, encompassing crucial process configurations such as segregation, IMDG codes, operators, and container types.

3Park is a 3D web application enabling real-time visualization of containers across all parks within the port. This application integrates with existing Terminal Operating Systems (TOS) of the port, meticulously monitoring all container movements and validating them in accordance with International Maritime Organization (IMO) standards for the segregation of containerized cargo.

Navplanning

NavPlanning platform facilitates the management of ship moorings in ports and the planning of arrivals and departures through an interactive map. This solution integrates with the Single Logistics Window, the official declaration system for announcing and processing procedures among port agents, and allows for real-time map visualization of ships using AIS signals.

Users can create plans and scenarios for allocating ships to port spaces, distributing the available port space for each day. This includes conducting depth studies (ship draft), assessing the mooring strain on bollards (angles), and ensuring proper spacing between moorings.

Shorex

3Shorex platform constitutes a spatial data infrastructure designed to consolidate the geographical information of clients (parking spaces) into a single database. This solution allows for managing port accesses related to both cruise activities (shorex entries) and non-shorex activities (port personnel, cargo handling). It also enables the management of configurations associated with the process, such as schedules, types of operators, port users, parking spaces, vehicle types, and activity logging.

The platform facilitates the management of numerous cruise ships visiting the port regularly by organizing all entries that provide services to ships or passengers, including taxis, bus shuttles, private transports, and water/goods supplies. It grants authorization to each vehicle or person entering the port, streamlining what would otherwise be a chaotic process.

3Shorex allows port authorities to issue entrances or approve entry requests sent by service providers. It utilizes QR codes for security validation at the port entrance. All validated entries are displayed on a map, providing a real-time view of the space being occupied.

Todoist

Todoist is a web-based Task Management Dashboard developed using Node.js, Vue 3, and Vuetify. This platform enhances team collaboration through real-time task tracking and intuitive visualization. It features a responsive design with real-time updates via websockets, allowing for seamless task organization and user-specific task views.

The system allows the registry of users, and enables them to create, assign, define and prioritize tasks.
